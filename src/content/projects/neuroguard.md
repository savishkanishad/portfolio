### Project Overview
NeuroGuard was built to solve the critical issue of road safety. Using computer vision, it monitors facial landmarks to detect signs of drowsiness.

### Key Features
*   **Real-time Detection:** Uses a webcam feed to analyze eye-blink frequency.
*   **Alert System:** Triggers an alarm when distraction levels exceed the threshold.
*   **Secure Logs:** Uses SQLite to log events for later review.
