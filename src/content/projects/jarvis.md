### Project Overview
Jarvis is a high-end desktop AI assistant designed for seamless interaction and productivity. It combines the power of large language models with native system capabilities to create a truly "Jarvis-like" experience on Windows.

### Key Features
*   **Intelligent Chat:** Integrated with Mistral-7B via HuggingFace for natural conversations and assistance.
*   **System Automation:** Control Windows apps, search the web, and get real-time system diagnostics through voice-like commands.
*   **Productivity Suite:** Built-in task manager, notes system, reminders, and a persistent memory system.
*   **Premium Interface:** A custom-built Tkinter GUI featuring chat bubbles, sidebars, and smooth animations.
